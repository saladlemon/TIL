## useEffect()

코드를 한 번만 호출하는 함수

```javascript
import{useEffect} from "react";

useEffect(한 번만 실행하고 싶은 코드, 배열);
```

ex)

```javascript
const [counter, setValue] = useState(0);
const onClick = () => setValue((prev) => prev + 1);
console.log("I run all the time");
useEffect(() => {
  console.log("CALL THE API...");
}, []);
```
